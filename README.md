# AI-Based Cyber Threat Detection System (FYP)
**An Intelligent Network Intrusion Detection System (NIDS) using Machine Learning**

Is project ka main maqsad network traffic ko real-time mein analyze karna aur malicious activities (Cyber Threats) ko detect karna hai. Ye system Machine Learning algorithms ka use karte hue traffic anomalies ko high precision ke sath classify karta hai.

---

### Features
* **Real-time Traffic Monitoring:** Network traffic ko analyze kar ke anomalies aur threats ko identify karta hai.
* **Multi-Class Classification:** Diffent types ke attacks (e.g., DoS, DDoS, Probe, R2L, U2R) ko classify karne ki salahiyat.
* **High Precision & Accuracy:** Model ko optimize kiya gaya hai taake false positives kam hon aur detection accuracy behtar ho.
* **Data Visualization:** Seaborn aur Matplotlib ke zariye network patterns aur attack correlations ka detailed analysis.

---

### Tech Stack
* **Language:** Python 3.x
* **Libraries:** Scikit-Learn, Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Algorithms:** Random Forest, SVM, Logistic Regression (as per project implementation)
* **Environment:** Jupyter Notebook / Google Colab
